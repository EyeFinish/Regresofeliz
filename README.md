# RegresoFeliz - Formulario de Reserva

Página web con formulario de reserva para el servicio de transporte RegresoFeliz.

## Características

- ✅ Formulario completo con validaciones
- ✅ Mapa interactivo con Leaflet y OpenStreetMap (100% gratuito)
- ✅ Autocompletado de direcciones con Nominatim
- ✅ Cálculo automático de distancia y costo
- ✅ Precio base de $25.000 + $500 por kilómetro
- ✅ Diseño responsive y moderno

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
- Leaflet.js
- OpenStreetMap
- Nominatim API
- OSRM (Open Source Routing Machine)

## Instalación

1. Clona este repositorio
2. Abre `index.html` en tu navegador
3. ¡Listo! No requiere configuración adicional ni API keys

## Estructura del Proyecto

```
felizregreso/
├── index.html      # Estructura del formulario
├── styles.css      # Estilos y diseño
├── script.js       # Lógica y funcionalidad
└── README.md       # Este archivo
```

## Uso

1. Completa los datos personales
2. Escribe el centro de evento y destino final (mínimo 3 caracteres)
3. Selecciona las ubicaciones de las sugerencias
4. El mapa mostrará la ruta y calculará automáticamente el costo
5. Completa el resto del formulario y envía
